This is the workflow a single post runs through when someone checks it. They submit a post, and the first branch is the input type — a Bluesky link we fetch, a block of text we parse, or a screenshot we read with OCR and vision.

All three converge: we extract the claim candidates, then retrieve similar evidence from the news corpus. The second decision is the important one — did we find supporting sources? If yes, we corroborate against them; if no, we flag it as low confidence rather than pretend.

Either way it generates the reader report — sources, trust signals, red flags — and, as always, no truth verdict.
